Effective ATS: An Array-Based Stack Implementation

ATS advocates a programming paradigm referred to as Programming with Theorem-Proving (PwTP) in which programs and proofs are syntactically intertwined. As proof construction can be highly expensive in terms of both effort and time, making proper use of unsafe programming features (so as to avoid explicity proof construction) is often a necessary skill for a programmer who wants not just to be able to code in ATS but to do so productively.
This article is written by Hongwei Xi.
